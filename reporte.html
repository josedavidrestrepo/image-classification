Informe No. 3

Santiago Alvarez, José Aristizabal, Angie López, Helber Padilla, José David Restrepo y Karen Rivera. 25 de febrero del 2019

Esta aplicacion se hizo con el objetivo para que cualquier persona tenga la posibilidad de interactuar con diferentes imagenes del data set, cada una de estas imagenes poseen caracteristicas diferentes, el modelo podra predecir si el sujeto lleva gafas oscuras o no, que es el fin ultimo para el cual el modelo se entreno.

REQUISITOS DEL CLIENTE

Construir y validar un modelo de aprendizaje estadistico para clasificar imagenes de sujetos con gafas oscuras de un data set dado.

Procesamiento de la imagen

La imagen cargada primero se reducirá a un tamaño de 60x64 pixeles, luego se pasará a una escala de grises, estas dos transformaciones se hacen para que su procesamiento sea mucho más rápido ya que necesitaría menos recursos computacionales; ya despues de esto la imagen se introduce en el modelo para ser procesada.

Modelo

En nuestro modelo utilizamos una red neuronal convolucionada, esto se decidio asi ya que el tiempo de procesamiento es menor por lo que el resultado se mostrará más rápido, además se utilizaron las librerías Keras y EBImage.

Primero que todo se leen todas las imágenes y se mandan a una lista,luego se leen sólo las imágenes en las que salgan personas con gafas oscuras y se mandan a otra lista diferente; después se cambia el tamaño de las imágenes de las dos listas al tamaño estándar 60x64 píxeles y en escala de grises esto se hace para poder meter las imagenes al modelo; el modelo básicamente está compuesto de cuatro capas de convolución y dos capas de pooling, por ultimo esta la parte en la que se entrena el modelo y se evalua la predicción de este.
